# Recipe Book

A modern, beautiful recipe book website built with React and Vite.

## Features

- Browse and view recipes
- Beautiful, responsive UI
- Fast and modern development experience

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

The app will open at `http://localhost:3000`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Tech Stack

- React 18
- Vite
- React Router
- Modern CSS
